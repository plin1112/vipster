New C++ version

Due to some architectural difficulties with the first python-based incarnation, I decided to rewrite Vipster in C++. Unfortunately, still quite some functionalities are missing so for productive use, please consider trying the old version first!

Thanks to emscripten and the brilliant Jonathan Gruber, this rewrite is available for you to try in your browser!

    Old python version

    The first version of vipster was written in Python. It is quite mature and feature-complete, but not devoid of bugs. Use it as a fallback for QM calculations, as it contains many features not yet reproduced in the rewrite. Unfortunately doesn't scale too well in the order of more than a few thousand atoms, so use with care when performing larger Lammps calculations!

    Depends on Python2.7/3, Numpy, PyQT4/5 and the Python OpenGL bindings.
